Bizzi — Démo bulle de chat

Page minimale de test pour bizzi-loader.js. La bulle apparaît en bas à droite.

Tenant simulé

Cette page monte le renderer chat avec data-bizzi-tenant="airbizness".

Endpoint API : https://bizzi.fr/api

Intégration côté tenant

<script src="https://bizzi.fr/bizzi-loader.js" defer></script>
<div data-bizzi-mount="chat"
     data-bizzi-tenant="airbizness"></div>

Attributs supportés

  • data-bizzi-mount="chat" — type de renderer (obligatoire)
  • data-bizzi-tenant="..." — slug du tenant (obligatoire)
  • data-bizzi-api="..." — override de l'URL API (optionnel)
  • data-bizzi-title="..." — titre du panneau (optionnel)
  • data-bizzi-color="#hex" — couleur primaire (optionnel)
